## ケース1：URLからテキスト抽出→ラベリング→scan保存

### フロー
```
URLをurls_<種別>.txtに記入 → url_batch.py実行 → scan/に自動保存
```

### 手順

**① URLファイルに記入**（初回のみ自動作成される）
```
corpus/phase2/urlfile/urls_web.txt   # Web記事・HP
corpus/phase2/urlfile/urls_note.txt  # noteプラットフォーム
corpus/phase2/urlfile/urls_sn.txt    # SNS
corpus/phase2/urlfile/urls_ad.txt    # 広告・商材
```
1行1URL。#で始まる行はスキップ。

**② 一括取得・自動ナンバリング・scan保存**
```powershell
python scripts\url_batch.py web    # urls_web.txtを処理
python scripts\url_batch.py note   # urls_note.txtを処理
python scripts\url_batch.py ad     # urls_ad.txtを処理
python scripts\url_batch.py sn     # urls_sn.txtを処理
```
→ corpus/phase2/scan/ に web138.txt, web139.txt... と自動保存される。

**③ スキャン**（ケース共通：後述）

## ケース4：human_label・riskfactor入力（xlsm上）

### T列（human_label）ショートカット
```
1 → HIGH
2 → MEDIUM
3 → LOW
4 → （空白）
```

### U列（riskfactor主因）ショートカット
```
1  → No academic/empirical support
2  → Potential concealment of adverse info
3  → Excessive promotion of gambling nature
4  → Offshore diversion (no legal protection)
5  → Safety claims inconsistent with reality
6  → Emotional Induction
7  → Desire Activation
8  → False Positive
9  → False Negative
10 → Intent-Unresolved
11 → Authority Halo
12 → Dissent Suppression
```

### V列・W列（riskfactor追加要素）ショートカット
U列と同じショートカット番号を使用。複合リスクを記述する。
例：U=6（Emotional Induction）、V=7（Desire Activation）
```
0 → 空白クリア
1〜12 → U列と同じ
```

### 既存列に一括適用する場合
```
対象列を範囲選択 → 開発タブ → マクロ → 「一括変換実行」→ 実行
```

### D：混同行列を確認（閾値41）
```powershell
python -c "
import pandas as pd
df = pd.read_csv('data/results/corpus_snapshot.csv')
df = df[df['human_label'].isin(['HIGH','MEDIUM','LOW'])]
tp = ((df['cmi']>=41)&(df['human_label']=='HIGH')).sum()
fp = ((df['cmi']>=41)&(df['human_label']!='HIGH')).sum()
fn = ((df['cmi']<41)&(df['human_label']=='HIGH')).sum()
tn = ((df['cmi']<41)&(df['human_label']!='HIGH')).sum()
pr = tp/(tp+fp)*100 if (tp+fp)>0 else 0
rc = tp/(tp+fn)*100 if (tp+fn)>0 else 0
f1 = 2*pr*rc/(pr+rc) if (pr+rc)>0 else 0
print(f'TP={tp} FP={fp} FN={fn} TN={tn}')
print(f'Precision={pr:.1f}% Recall={rc:.1f}% F1={f1:.1f}')
"
```
